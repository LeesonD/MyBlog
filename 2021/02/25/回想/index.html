回想

发表于 2021-02-25 | | 阅读次数:

2月23号收到域名到期邮件时才回想起来还有这个博客，趁今晚有空补一篇。
也快忘了怎么用hexo，在这一起回想下，如果两年后又忘了怎么用就不用再查了!

本地hexo目录下执行 $hexo new [layout] <title>
这样就可以编辑md文件了！  
编辑完后一键部署方法：
同目录下执行 $hexo deploy
如果没有反应先执行一下 $hexo clean 再执行deploy

部署期间遇到一个坑，在_config.yml中：后面要加空格，不然笨蛋hexo不识别

这样就完成了备忘录，再过两年也能流畅发博了~
